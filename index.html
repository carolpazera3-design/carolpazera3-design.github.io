API & Integrasi

REST API

Endpoint sederhana dengan JSON. Otorisasi menggunakan bearer token. Webhook untuk status transaksi.

POST /v1/orders
Authorization: Bearer <API_KEY>
{
  "product_id": "pln-prepaid",
  "customer_no": "5489xxxxxxx",
  "ref_id": "INV-2025-0001"
}

Keamanan

  • Rate limiting per API key
  • Signature opsional HMAC
  • Whitelist IP dan rotasi key
